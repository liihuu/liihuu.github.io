# liihuu.me

Personal homepage for liihuu, a frontend engineer focused on financial frontend engineering and the author of KLineChart.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

The site is configured for GitHub Pages with the custom domain `liihuu.me`.

```bash
npm run deploy
```

`public/CNAME` is copied into `dist` during build so GitHub Pages keeps the custom domain mapping.
